BACK.

This web system is designed to be used as a “task-management” system in the form of a to-do list.

When users visit the app they can see current errands that are already outstanding, and see if they have been completed or not by seeing if they have been crossed through or not, by selecting a specific outstanding errand they can cross it out and mark it as complete. In addition to this, users can add their own errands, as well as edit or delete already existing ones.

The API is built for this web system using Django Rest API
